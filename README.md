# Canary 🐤⛏️  
*A digital canary for modern mining safety*

## 🚨 Elevator Pitch
A digital canary for mines—detecting dangerous gases in real time and mapping their spread to prevent accidents and save lives.

---

## ⚙️ Tech Stack
- Raspberry Pi  
- Python  
- GPIO  
- MQ-series gas sensors  
- DHT11 humidity sensor  
---

## 🤝 Team
Built during YHack to explore low-cost, real-time safety systems for hazardous environments.
